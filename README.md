Pokédex App

Aplicación web desarrollada con React que consume la API pública de Pokémon para mostrar información detallada de cada Pokémon, incluyendo estadísticas, tipos, debilidades y evoluciones.

 Tecnologías utilizadas
React
React Router
JavaScript (ES6+)
CSS
API: https://pokeapi.co/
 Funcionalidades
 Búsqueda de Pokémon
 Visualización de estadísticas con barras
 Tipos y debilidades dinámicas
 Navegación entre Pokémon (prev / next)
 Cadena de evoluciones
 Descripción del Pokémon desde species
 Uso de Inteligencia Artificial

Se utilizó IA como apoyo durante el desarrollo, principalmente para resolver dudas y mejorar algunas partes del código.

No se usó para construir todo el proyecto, sino más como una herramienta de ayuda en momentos específicos.

Uso de Inteligencia Artificial

Durante el desarrollo del proyecto se utilizó IA como herramienta de apoyo para agilizar el proceso de codificación y resolver dudas puntuales.

Su uso fue principalmente como asistente, proporcionando sugerencias a partir de instrucciones concretas que yo definía previamente (por ejemplo, cómo estructurar ciertas llamadas a la API o cómo representar datos en la interfaz). Es decir, la lógica y el enfoque del proyecto fueron planteados por mí, y la IA se utilizó para acelerar la implementación de algunas partes.

No se utilizó para construir la aplicación completa, sino como apoyo en momentos específicos donde era útil validar ideas, corregir errores o mejorar la claridad del código.

 ¿Cómo se utilizó la IA?
Para agilizar la escritura de código a partir de ideas ya definidas
Resolver errores puntuales durante el desarrollo
Apoyar en la estructura de algunas funciones (como consumo de API y manejo de datos)
Sugerencias para mejorar estilos y organización del código
Procesamiento del código

Todo el código generado con apoyo de IA fue revisado, adaptado y comprendido antes de integrarse al proyecto.

Se tiene claro el funcionamiento de:

La obtención de datos desde la API (Pokémon, species, tipos, evoluciones)
El uso de useEffect para ejecutar las llamadas cuando cambia el id
El manejo de estado con useState
La construcción de la lógica para debilidades y evoluciones
El renderizado dinámico de la información en la interfaz
